ASCIIper is an experimental, ASCII-based dungeon management game inspired by the classic Dungeon Keeper. It was created primarily to explore the frontiers of "vibe coding" using Google Antigravity and Gemini.

This project serves as a hands-on exploration of agent-first development. Instead of writing every line of code manually from scratch, ASCIIper was built by directing autonomous AI agents within the Google Antigravity IDE. The goal was to test how effectively an AI-driven workflow can handle the logic, rendering, and entity management of a fully playable text-based simulation.

⛏️ Features
Classic Dungeon Management: Carve out your underground empire, build specialized dungeon rooms, and manage your subterranean ecosystem.

Minion Command: Order around your loyal imps and goblins to mine resources, claim territory, and maintain the dungeon.

Retro ASCII Aesthetic: A fully text-based interface that leaves the graphical heavy lifting to your imagination.

AI-Generated Architecture: The underlying code structure was orchestrated using Antigravity's multi-agent execution model.

💻 Tech Stack & Vibe Coding
Google Antigravity: Utilized the Agent Manager to asynchronously plan and execute the game's core loops, state management, and entity behaviors.

Gemini: The driving intelligence behind the code generation, problem-solving, and implementation plans.

Vibe Coding Methodology: The development process focused on high-level prompting, reviewing AI-generated artifacts, and iterating on the design through natural language rather than manual typing.
